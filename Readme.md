>mysql workbench로 모델링한 내용을 sqlite데이터베이스에 적용시켜 .db파일로 추출함

>phpmyadmin, db browser for sqlite, mysql workbench

- .mwb 는 workbench로 모델링한 파일
- .sql 는 phpmyadmin에서 데이터베이스에 적용된 쿼리 내용들
- .db 는 sqlite로 모델링된 파일
